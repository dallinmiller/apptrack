# AppTrack

AppTrack is a program that stores, creates alerts, and analyzes data for a user currently submitting job applications. 

## Features
- Collects and stores application data in CSV files
- Basic statistical analysis for insights on application success
- Notification system for milestones and deadlines.

## How to Run
1. Clone the repo
2. Run AppTrack

## Project Structure
- alerts/ - notification and updates
- process/ - main logic and data handling
- menus/ - simple ui programs
- classes/ - data setup
- data_files/ - data storage
